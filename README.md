# 🚀 DigiTools Platform

DigiTools Platform is a website where people can buy different digital tools all in one place. Users can also subscribe to get access to all tools for a certain time and enjoy special offers.

- HTML

- DaisyUI  
- JavaScript  
- React.js  
- JSON  

---

## ✨ Main Features

- 🛒 Add products to your cart  
- ❌ Remove products from your cart  
- 💳 Checkout products easily  
- 📦 Subscribe to access all tools  
- 🔔 Shows notifications when you add/remove items

## 🛠️ How to Run Locally

1. **Clone the project**
```bash
git clone https://github.com/Farhan1471/digitools-platform.git
```

2. **Go to project folder**
```bash
cd digitools-platform
```

3. **Install dependencies**
```bash
npm install
```

4. **Start the project**
```bash
npm run dev
```

5. **Open in browser**
```
http://localhost:5173
```
